# Описание - Анализ доходов и расходов физ.лица

## Данные

В наличии были следующие данные о доходах и расходах за полтора года:
- Дата и время - дата операции
- Категория	- категория поступления/траты
- Счет - с какого счета была списанана/зачислена сумма
- Сумма	- размер поступления/траты
- Валюта - валюта операции

## Задача

Проанализировать и проиллюстировать данные, найти закономерности и особенности. Сравнить периоды до и после переезда.

## Используемые библиотеки
*Pandas*, *Matplotlib*, *Plotly.express*, *Seaborn*
